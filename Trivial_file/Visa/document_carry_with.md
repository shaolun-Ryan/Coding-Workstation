# 面谈原则
* 无移民倾向(国内发展好、独生子)
* 有能力完成学业
* 有钱完成学业

## 材料模板
### 学习计划
* 简要介绍自己的情况、本科学校等
* 学校和program情况，夸夸学校
* 其他申请的学校(不用写太多，就说刚开始申请就和导师取得了联系)
* 归国计划
  
## 面谈
* 走到窗口，简短的自我介绍：谁，去哪儿，干嘛，完事儿呢？
* 熟悉每一份材料的作用、顺序、需要时及时拿出（或者作出动作）
* 每个问题，回答后主动拿出（For more details you can see my XXX）
* 面谈内容
